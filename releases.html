Version 0.1.19 - Feb 2015

Fix for Mailtrack incompatibility and new Features.

  1. Extension is now compatible with Mailtrack
  2. Added mark as read. Experimental, can sometimes leave the "More" window open when using

Version 0.1.18 - Dec 2014

Functionality removal

Labeling was not working from release 0.1.16 so removed so the break fix 0.1.17 could be released

Version 0.1.17 - Oct 2014

Break Fix

Chrome was causing the Window.event object to be null and was causing the script to crash.

Version 0.1.16 - July 2014

New Functionality

Added "Mark As Spam" and "Label As" functionality

Version 0.1.14 - June 2013

Fix - Google implemented the click action.

Previously the button click on a row fired on a mouse down/mouse up combination. In late May 2013, Gmail™ changed to work off a click action. This action was firing when clicking on one of the "Actions for Gmail™" buttons, so rather than performing the "Actions for Gmail™" action, it was viewing the email.

This fix stopped the "click" action from propagating, when clicking on an "Action for Gmail™" button.

Version 0.1.10 - April 2013
Fix - Google changed the div elements around the checkboxes on each email row which caused the extension to stop working
